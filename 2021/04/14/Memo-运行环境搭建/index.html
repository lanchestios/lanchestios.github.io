集群模式

假设使用三台机器来搭建一个 ZooKeeper 集群。首先，我们假设已经准备好三台互相联网的 Linux 机器，它们的 IP 地址分别为 IP1、IP2 和 IP3：

  1. 准备 Java 运行环境。确保你已经安装了推荐版本的 JDK。

  2. 下载 ZooKeeper 安装包。

  3. 配置 zoo.cfg 文件（初次使用将conf/zoo_sample.cfg重命名为conf/zoo.cfg。

  4. 创建 myid 文件。

    在 dataDir 所配置的目录下，创建一个名为 myid 的文件，在该文件的第一行写上一个数字，和 zoo.cfg 中当前机器的编号（也就是 server.id 中的 ID 号，ID 号取值范围为 1~255）对应上。

  5. 为每个服务器配置相同的内容。

  6. 启动服务器。

集群模式下，节点 Mode 属性为 Leader 或 Follower。
